## ✨ Features

- 🎨 **Dark Glassmorphism Design** — frosted glass cards, glowing accents, animated background mesh
- 🌙 **Persistent Light/Dark Mode** — theme saved across all pages using `localStorage`
- 📱 **Fully Responsive** — mobile-first design with breakpoints at 600px and 900px
- 📐 **CSS Grid & Flexbox** — used throughout for layout and alignment
- 🗃️ **CSS Custom Properties** — all colours and styles controlled via CSS variables
- ✅ **To-Do App** — full CRUD, `localStorage` persistence, and All/Active/Completed filtering
- 📬 **Working Contact Form** — emails delivered via Formspree
- ♿ **Accessible** — semantic HTML, ARIA labels, keyboard-navigable

## 📁 Project Structure

```
portfolio/
├── index.html        # Home page
├── about.html        # About page
├── projects.html     # Projects page
├── contact.html      # Contact page
├── todo.html         # To-Do App page
├── style.css         # Main stylesheet (glassmorphism theme)
├── todo.css          # To-Do App specific styles
├── theme.js          # Dark/light mode toggle with localStorage
└── todo.js           # To-Do App logic (CRUD, filtering, localStorage)
```
